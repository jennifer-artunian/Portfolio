# Air Quality Data Analysis – Particulate Matter & Temperature Trends

## 📊 Overview

This project analyzes the relationship between air particulate matter (PM) levels and temperature using real-world environmental data. The goal is to identify patterns, trends, and potential correlations that can provide insights into air pollution behavior.

---

## 🎯 Objective

- Analyze trends in air pollution data  
- Explore the relationship between temperature and particulate matter levels  
- Extract meaningful insights from environmental datasets  

---

## 🛠️ Tools & Technologies

- Python (Jupyter Notebook)  
- Pandas
- GeoPandas
- NumPy  
- Matplotlib / Seaborn  

---

## 📈 Key Analysis

- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Correlation analysis between temperature and PM levels  
- Visualization of trends over time
