# Air Quality Data Analysis – Particulate Matter & Temperature Trends

## 📊 Overview
This project analyzes the relationship between air particulate matter (PM) levels and temperature using real-world environmental data. The goal is to identify patterns, trends, and potential correlations that provide insights into air pollution behavior.

## 🎯 Objective
- Analyze trends in air pollution data  
- Explore the relationship between temperature and particulate matter levels  
- Extract meaningful insights from environmental datasets  

## 🛠️ Tools & Technologies
- Python (Jupyter Notebook)  
- Pandas, GeoPandas, NumPy  
- Matplotlib / Seaborn  
- Scikit-learn (Machine Learning)  

## 📈 Key Analysis
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Correlation analysis between temperature and PM levels  
- Basic predictive modeling using Scikit-learn  
- Visualization of trends over time  

## 📎 Project Files
- Jupyter Notebook (.ipynb)  
- PDF summary of analysis  

## 💡 Key Insights
- Identified patterns between temperature fluctuations and particulate matter levels  
- Observed trends that may indicate environmental and seasonal effects  
- Demonstrated ability to analyze and interpret real-world datasets  

## 🚀 Skills Demonstrated
- Data cleaning and preprocessing  
- Exploratory data analysis (EDA)  
- Data visualization  
- Analytical and statistical thinking
