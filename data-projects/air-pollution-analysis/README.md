# Air Pollution Data Visualization – Canadian Provinces

## Overview
This project analyzes air pollution data across Canadian provinces using Power BI to identify trends, geographic patterns, and environmental risk factors. The dashboard enables interactive exploration of pollution levels by province and time.

## Objectives
- Analyze air pollution indicators across Canadian provinces
- Identify temporal and geographic trends
- Develop an interactive dashboard for data-driven decision-making
- Demonstrate data visualization and analytical capabilities

## Tools Used
- Power BI
- Data cleaning and transformation
- Data modeling
- Interactive dashboard design

- presentation.pptx — Project presentation
