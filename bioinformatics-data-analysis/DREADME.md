# Bioinformatics Data Analysis – Small RNA Expression

## 🔬 Overview
This project focuses on the analysis of small RNA sequencing data (RNA-seq) to identify differentially expressed heterochromatic siRNAs (het-siRNAs) and phasiRNAs in the symbiotic interaction between *Phaseolus vulgaris* and *Rhizobium etli*.

The analysis combines bioinformatics pipelines and statistical methods to extract meaningful biological insights from high-throughput sequencing data.

---

## 🎯 Objective
- Identify differentially expressed small RNAs
- Analyze regulatory patterns in plant–bacteria symbiosis
- Extract data-driven insights from large biological datasets

---

## ⚙️ Methodology
- Adapter trimming and preprocessing of sequencing data  
- Filtering sequences by size (21–24 nucleotides)  
- Removal of rRNA and tRNA sequences  
- Alignment using Bowtie2  
- Differential expression analysis  
- Identification of phasiRNAs  
- Target prediction and functional annotation  

---

## 📊 Key Results
- Identified **236 differentially expressed het-siRNAs**  
- Detected **678 differentially regulated phasiRNAs across 18 loci**  
- Found associations between small RNAs and gene regulation  
- Observed inverse expression patterns between RNAs and their targets  

---


## 🔄 Data Pipeline

Designed and executed an end-to-end bioinformatics pipeline for RNA-seq data analysis:

1. Raw data acquisition from high-throughput sequencing libraries  
2. Adapter trimming using Trim Galore  
3. Sequence filtering and preprocessing (FastQ)  
4. Alignment to reference genome using Bowtie2  
5. Differential expression analysis using DEUS (R, DESeq2)  
6. Data visualization using ggplot2 to explore expression patterns  
7. Functional annotation and target identification  

This pipeline enabled efficient processing, analysis, and interpretation of large-scale biological datasets.
