# Piping and Instrumentation Diagram (P&ID)

This section contains the Piping and Instrumentation Diagram developed for a key process unit of the ammonium sulfate production plant.

This diagram includes:

- piping connections between equipment
- instrumentation and control loops
- control valves and safety devices
- measurement systems (flow, temperature, pressure, and level)

## Included Diagram

- **R101 Reactor P&ID**  
  Detailed instrumentation and control scheme for the main stirred reactor, including temperature, pressure, level, and flow control loops.
