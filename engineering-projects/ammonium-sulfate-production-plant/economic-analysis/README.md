# Economic Analysis

This section presents the economic evaluation of the proposed ammonium sulfate production plant.

Financial feasibility was assessed through capital investment estimation, operating cost analysis, profitability indicators, and break-even analysis.

## Contents

- Capital Expenditure estimation (CAPEX)
- Operating cost evaluation (OPEX)
- Break-even point analysis
- Net Present Value (NPV / VAN)
- Internal Rate of Return (IRR / TIR)
- Sensitivity analysis

The analysis integrates CAPEX and OPEX estimations to determine the economic feasibility and profitability of the project under different market conditions.
